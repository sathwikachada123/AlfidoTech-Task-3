Overview of the Project:

Project:Instagram Data Analysis

Introduction: Instagram is one of the most popular social media platforms, producing large amounts of engagement data.

This project focuses on analyzing Instagram datasets to identify posting trends, content performance, and follower 

engagement signals using data science techniques.

Objective:

To analyze Instagram datasets & uncover insights about:Best posting times (days & hours)Content types with higher

engagement(photo, video, carousel) Verified vs non‑verified user behavior ,Follower growth signalsHashtag

effectiveness.

Dataset Description:

The project uses multiple datasets including:Users (profile info, verified status, post counts)Photos (timestamps,
 
content type)Likes (user interactions)Comments (text, hashtags, timestamps)Followers (relationships, activity status)

Tags(hashtagsand tag IDs).

Methodology

1.Data Preprocessing
   
Standardized column names, parsed dates, handled missing values.

Extracted features like day of week and posting hour.

2.Exploratory Data Analysis (EDA)

Distribution plots for likes, comments, hashtags, verified vs non‑verified users.

Content type analysis (photo, video, carousel).

3.Engagement Metrics

Calculated likes/comments per follower.

Computed engagement rate across verified and non‑verified accounts.

4.Visualization

Bar plots, histograms, scatter plots for posting trends and engagement.

5.Feature Importance

Studied how tag density and posting behavior influence performance.

Results:

Morning hours (7–11 AM) show peak posting activity.

Friday & Monday are top‑performing days.

Photos dominate engagement compared to videos/carousels.

Top hashtags significantly boost visibility.

Verified vs Non‑Verified: Non‑verified users are more numerous, but verified accounts maintain stronger trust signals and higher engagement quality.

Engagement rate varies with follower count; smaller accounts often show higher relative interaction.

Conclusion:

This project demonstrates the practical use of data science in analyzing social media engagement. The insights guide 

businesses and creators in optimizing posting schedules, content types, and hashtag strategies to maximize reach and 

interaction.

Technologies Used: Python,Pandas,NumPy,Matplotlib,Seaborn,Jupyter Notebook.
